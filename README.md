# YogiBeani

Charlene's Yoga Studio — class scheduling, Stripe payments, waivers, and client management.

## Quick Start

```bash
cd backend && npm install
node server.js
# Open http://localhost:3000
```

## Structure

- `frontend/` — Vanilla JS website (HTML, CSS, JS)
- `backend/` — Express + SQLite API server
- `claude-agents/` — Shared Claude agent configs (git submodule)
